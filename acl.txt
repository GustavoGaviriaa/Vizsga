LAN3 ne erje el LAN2-t de LAN1-et igen

Laptop2 a servert ne erje el http-vl, https-el es minden massal igen, mast ne korlatozzunk

laptop1 ne tudja pingelni a laptop0 at es csak ftp-vel erje el a servert

R1(config)
access-list 10 permit 192.168.20.10 0.0.0.0
access-list 10 deny any
line vty 0 4
access-class 10 in
transport input telnet
login local


access-list 100 deny ip 192.168.30.0 0.0.0.63 192.168.20.0 0.0.0.127
access-list 100 permit ip 192.168.30.0 0.0.0.63 192.168.10.0 0.0.0.255

interface g0/0/1 
ip access-group 100 in


ip access-list extended MAU
deny tcp 192.168.30.0 0.0.0.63 192.168.10.0 0.0.0.255 eq 80

permit tcp host 192.168.20.10 host 192.168.10.254 eq 21
deny ip host 192.168.20.10 host 192.168.10.254
permit ip any any
interface g0/0/0
ip access-group VAU in
